# ChronoAtlas

一个交互式世界历史年表项目，第一阶段聚焦公元 180-280 年：

- 中国：东汉崩溃与三国形成
- 地中海：罗马帝国三世纪危机
- 西亚：萨珊波斯取代安息帝国
- 印度次大陆：贵霜后期与区域政权变化

项目目标不是做普通百科，而是回答一个更直观的问题：

> 同一时间，世界不同地方正在发生什么？

## First MVP

第一版先做一个小而完整的交互原型：

1. 拖动时间轴，选择 180-280 年之间的年份或时间段。
2. 地图或区域列表显示同时间段的关键事件。
3. 点击事件卡片，展示简述、时间、地区、人物、前因后果、关联事件和来源。
4. 支持横向对照，比如“公元 220 年，中国、罗马、萨珊波斯、印度次大陆各发生了什么？”

## Why This Slice

公元 180-280 年很适合作为起点：

- 中国从东汉走向三国，叙事强、用户熟悉。
- 罗马同时进入三世纪危机，能形成强烈的东西对照。
- 萨珊波斯在 224 年兴起，是连接罗马、印度和中亚的重要力量。
- 印度次大陆处于贵霜后期和区域重组阶段，能避免世界史只看中西两端。

## Repository Map

- `docs/product-brief.md`: 产品方向和核心体验。
- `docs/mvp-scope.md`: 第一阶段范围、边界和优先级。
- `docs/data-model.md`: 历史事件数据结构。
- `db/seeds/`: SQLite 种子数据，当前核心人物、事件、来源和证据链从这里重建。
- `db/migrations/`: SQLite schema 迁移。
- `scripts/history-api-server.mjs`: 前端读取 SQLite 的本地 API。
- `data/china-three-kingdoms-map.json`: 暂保留的地图数据，后续地图重做时再迁移。
- `src/`: 第一版交互原型。
