# MVP Scope

## Time Range

公元 180-280 年。

这个范围允许第一版覆盖一个完整的历史转折：

- 东汉末年到三国鼎立。
- 罗马从康茂德时期走向三世纪危机。
- 萨珊波斯从兴起到成为罗马主要东方对手。
- 印度次大陆从贵霜影响力衰退走向区域化。

## Regions

第一版只保留四个主区域：

- `china`: 东汉末年、魏蜀吴、晋统一前夜。
- `rome`: 罗马帝国，尤其是三世纪危机。
- `sasanian-persia`: 安息末期与萨珊波斯。
- `india`: 印度次大陆，贵霜后期和地方政权。

## Event Types

第一版优先录入这些类型：

- `politics`: 政权更替、皇帝即位、权力斗争。
- `war`: 战争、战役、边境冲突。
- `society`: 起义、迁徙、社会变化。
- `culture`: 宗教、思想、文化传播。
- `economy`: 贸易、货币、财政和生产变化。

## Non-Goals For MVP

- 不做全世界历史覆盖。
- 不做精细 GIS 历史地图。
- 不让 AI 直接生成未经审核的史实入库。
- 不追求每一年都有事件。
- 不做复杂账号系统。

## First Build Target

做一个网页原型：

- 顶部或底部时间轴。
- 中间区域对照面板。
- 右侧事件详情。
- 年份输入框和前后跳转。
- 简单搜索。

地图可以第二阶段再做，第一版用四个区域列就足够验证体验。
