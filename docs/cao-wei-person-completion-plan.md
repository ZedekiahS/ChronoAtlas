# 曹魏人物补全计划

目标不是一次性堆满人名，而是把《三国志》魏书与《资治通鉴》中的曹魏相关人物整理成可审校、可追踪、可扩展的人物资料库。

## 补完标准

一个人物进入“可用档案”至少需要：

- `china-persons.json` 中有稳定 `id`、姓名、字号、生卒或生卒未详、主要政权、角色和摘要。
- 至少有一条《三国志》或《资治通鉴》来源；有本传者优先挂本传。
- 核心人物有 4 条以上生平节点，边缘人物至少保留来源化提及或待补状态。
- 参与重大事件的人物必须回链 `events-180-280.sample.json`。
- 明确的人物关系进入 `china-person-relations.json`。
- 未逐段核对裴松之注和《资治通鉴》前，`coverageStatus` 保持 `partial`，不标成完成。

## 推进顺序

1. 《三国志》魏书本传人物建档。
2. 《资治通鉴》卷五十八至卷八十一按年互校。
3. 裴松之注中的《魏略》《世语》《九州春秋》等扩展人物与异文。
4. 把同名、字号、官称、谥号、爵号逐步整理进别名体系。
